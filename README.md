# QuestAutomation

This project contains automated UI tests for https://automationexercise.com/ using Playwright and C#.

## Test Scenarios

- Login with invalid credentials
- Verify Test Cases page
- Verify products page and product details
- Search for a product
- Subscribe from home page
- Subscribe from cart page
- View category products
- View brand products
- Scroll up using arrow button
- Scroll up without arrow button
- Login with empty credentials (new)

## Tech Stack

- C#
- NUnit
- Microsoft Playwright

## How to Run

Run the following command in terminal:

dotnet test
